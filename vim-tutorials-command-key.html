Navigating the Cursor
  1. h - to move the cursor left
  2. l - to move the cursor right
  3. k - to move the cursor top
  4. j - to move the cursor down
  5. 0(zero) - to move to the start of the line
How to exit in VIM
  1. q! - exit without saving the changes
  2. wq - exit and save changes
How to insert
  1. i - to insert text or characters before the cursor
  2. a - another way to insert text or characters after the cursor
How to jumped /word using counting process
  1. 2w - jumped 2 words
  2. 3w - jumped 3 words
How to delete

  1. x - to delete single character
  2. dw - to delete from the cursor up to the next word
  3. d$ - to delete from the cursor to the end of a line
  4. ce - to deletes the word and places you in Insert mode.

  5. c$ - to delete from the cursor to the end of a line and places you in Insert mode.
How to delete /word using counting process
  1. d2w - to delet 2 words
  2. d3w - to delet 3 words
How to delete line or multiple lines

  1. dd - delete single line
  2. 2dd - delete 2 lines
How to replace character/s
  1. r - to replace character/s
  2. p - paste
Search on VIM
  1. " / " - command or key to find or search word/s direction forward

  2. " ? " - command or key to find or search word/s direction backward
  3. n - same direction
  4. N - opposite direction
  5. % - to match (), {}, []
Substitute
  1. :s/old word/new word/g - change word
Motion on VIM
  1. v - to highlight or select the characters, word/s or line/s
  2. v + h - to select left
  3. v + j - to select down
  4. v + k - to select up
  5. v + l - to select right
Open Command on VIM
  1. o (lowercase) - open new line below the cursor
  2. O (uppercase) - open new line above the cursor
Copy and Paste on VIM
  1. y - to copy
  2. p - paste
